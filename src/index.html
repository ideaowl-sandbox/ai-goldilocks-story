\text{Information Entropy } = H(Z) \doteq \sum_{n=1}^{k} P(z_n) \log_{2} \frac{1}{P(z_n)}
For a random variable Z, return the average number of bits needed to encode its outcomes.
Random Variable Z_{coin} =\{z_{heads}, z_{tails}\}
Probabilities are P(z_{heads}) = P(z_{tails}) = 0.5
H(Z_{coin}) = [0.5 \log_{2} \frac{1}{0.5}] + [0.5 \log_{2} \frac{1}{0.5}] = 1 \text{ bit}
H(Z_{coin}) = 1 \text{ bit}

For a random variable Z, return the average number of bits needed to communicate an action to take.
Z_{A(s)} =\{z_{a_{up}}, z_{a_{down}}, z_{a_{left}}, z_{a_{right}}\}
Probabilities are uniform P(z_{a_{up}}) = P(z_{a_{down}}) = P(z_{a_{left}}) = P(z_{a_{right}}) = 0.25
H(Z_{A(s)}) = 4 \times [0.25 \log_{2} \frac{1}{0.25}] ] = 2 \text{ bits}
H(Z_{A(s)}) = 2 \text{ bits}
